Celulares_Ch
============

Tesis: Sistema de control de inventarios para tiendas de celulares.

Descripción
============
El sistema que se pretende realizar estará montado en diferentes sucursales del estado de hidalgo.
Éste sistema es únicamente para uso de los empleados y tendrá cambios en un futuro.

El presente sistema permitirá llevar un mejor control sobre los equipos que se han vendido, equipos entregados a garantía o reparación así como los equipos que ya fueron entregados a sus respectivos dueños.

El sistema será capaz de generar reportes de:
  * Equipos en garantía.
  * Equipos en reparación.
  * Equipos vendidos al día.
